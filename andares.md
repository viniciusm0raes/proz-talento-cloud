### Contagem com o laço de repetição WHILE
_________________________________________________
```
i = 20
print("--------Descendo--------")
while (i > 0): 
    if i != 13:   
        print(i)
    i = (i - 1)
print("---Chegamos ao térreo---")
```
_________________________________________________
### Contagem com o laço de repetição FOR
```
print("--------Descendo--------")
for i in range(20, 0, -1): 
    if i != 13:   
        print(i)
print("---Chegamos ao térreo---")
```
