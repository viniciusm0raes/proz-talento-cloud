Bancos de Dados - Módulo I

De acordo com os comandos aprendidos, programe códigos SQL para criar um banco de dados chamado ESCOLA e deixe-o pronto para o uso. Depois, pesquise qual é o comando utilizado para inserir uma tabela no banco de dados e siga as instruções:

  • 1. crie uma tabela chamada ALUNO;
  • 2. defina os atributos da tabela;
  • 3. adicione a chave primária de nome ID (identificador);
  • 4. adicione um atributo nome do tipo varchar;
  • 5. adicione um atributo e-mail do tipo varchar;
  • 6. adicione um atributo endereço do tipo varchar.
Ferramenta online utilizada (PostgreSQL): https://sqliteonline.com/

Código(s) SQL utilizado para criação do Banco de Dados de nome Escola

Criando uma tabela de produtos

            CREATE DATABASE db_escola;

A ferramenta online utilizada não permite a criação de DATABASE. Códigos apenas demonstrativos.

Selecionar a tabela criada para alterações, inclusões, etc.

            PostgreSQL (via psql)
            \c nome_do_banco_de_dados;
            \c db_escola;

            MySQL            
            USE nome_do_banco_de_dados;
            USE db_escola;

Criação da tabela de nome tb_aluno e atendimento dos itens solicitados

            CREATE TABLE tb_aluno(
                id_aluno SERIAL PRIMARY KEY,
                nome VARCHAR(20) NOT NULL,
                email VARCHAR(30),
                endereço VARCHAR(80) NOT NULL
                );

Inserindo dados na tabela tb_aluno

            INSERT INTO tb_aluno(nome, email, endereço)
            VALUES
                ('Anabel da Silva', 'anabel@mail.com', 'Rua 1, nº 2, Manaus-AM'),
                ('Flávia dos Santos', 'flavia@mail.com', 'Rua 2, nº 1, Salvador-BA'),

Visualizando dados da tabela tb_aluno

            SELECT * FROM tb_aluno;
