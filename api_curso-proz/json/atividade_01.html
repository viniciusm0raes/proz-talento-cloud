JSON - Atividade 01

Crie o arquivo produtos.json e dentro dele escreva uma estrutura de dados equivalente a um array de objetos, onde cada objeto representa um objeto com pelo menos as seguintes propriedades: id, nome, marca, categoria, preço, quantidade em estoque, se o produto está ou não disponível para pronta entrega. Preencha os dados com as informações de três produtos da sua escolha. Observação: é interessante escrever a string num arquivo com extensão .json no VSCode, pois o editor nos mostrará qualquer possível erro de sintaxe com um sublinhado vermelho.
